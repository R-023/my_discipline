# Discipline Tracker

Простой трекер ежедневной дисциплины.  
Фокус на **процессе**, а не на результате.

## Функции
- Отметить успешный день (соблюдение правил)
- Автоматический подсчёт цепочки дней подряд
- Минималистичный календарь
- Работает локально и в облаке

## Запуск локально

1. Установите Python 3.8+
2. Установите зависимости:
